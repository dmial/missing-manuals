# missing-manuals — заметки для агента

Репозиторий — коллекция «недостающих руководств» по техническим темам. Один файл — одна тема, формат markdown. Заметки потом цитируются как справочник, поэтому фактологическая точность критична.

## При работе с заметками — используй скилл `missing-manual-creator`

В нём живут принципы изложения (пирамида, близость термина и определения, контекстная глубина, лаконичность) и правило сверки фактов с официальной документацией. Не дублируй эти правила здесь — ходи в скилл.

## Кросс-ссылки

Заметки ссылаются друг на друга относительными ссылками (см. `mcp-protocol.md` → `api-protocols.md`). При добавлении нового материала проверяй, нет ли смежных, на которые стоит сослаться.
